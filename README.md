# ClientHub

Web app per la gestione di clienti costruita con React e TypeScript.

# Funzionalità
- Visualizzazione clienti in tabella con paginazione
- Ricerca per nome ed email
- Filtro per stato (attivo/disattivo)
- Aggiunta, modifica ed eliminazione clienti
- Modal dettaglio cliente
- Dashboard con KPI e grafico a linee
- Notifiche toast per ogni azione
- Dark mode
- Layout responsive e mobile-first

# Tech Stack
- React + Vite
- TypeScript
- React Router
- useReducer per la gestione dello stato
- Recharts per grafici
- React Hot Toast per le notifiche
